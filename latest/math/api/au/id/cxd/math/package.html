package math

The package au.id.cxd.math contains a set of operations for the following areas

- count : functions related to counting, such as choose and factorial operations - data : utility libraries for handling basic data formats such as reading text input from csv - definitions : a set of classes that are used in defining types and concepts used in the library - function : implementations for known functions and certain operations on functions such as integration - model : routine concerned with specific types of modelling - probability : functions relating to probability and statistics
