DiscreteBayes

class DiscreteBayes extends AnyRef

A simplistic approach to the bayes rule allowing the distributions to be combined in a manner of enumerating the possible values for the conditional variable

$$ P(B_i|A) = \frac{ P(A and B) } {P(A) } $$

For discrete variables this is the likelihood multiplied by the prior divided by the marginalised P(A) $$ P(A) = \frac{ P(A|B_i) \times P(B_i) } { \sum_{j=1}^n P(A|B_j)\times P(B_j) } $$

For continuous values it is the marginalised value for P(A) is the integral \int_{-\infty}^\infty P(A|B)P(B)
